EcoSpold 01 / 02

The converter contains XML files, style sheets, and schemas from the EcoSpold format version 1 and 2. EcoSpold is a free and open source LCA data format licensed under the EcoSpold Public License Version 1.0 (ESPL). A copy of the ESPL is available at http://www.ecoinvent.org/fileadmin/documents/en/espl.pdf.
